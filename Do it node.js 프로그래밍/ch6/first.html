nodejs를 이용해서 클라우드 서비스를 이용할 시 관계형 데이터베이스는 비용이 더 많이 들거나 거의 대부분이 유료이다. 그래서 nosql의 DB를 사용하는 경우가 많다. 6장에서는 몽고디비를 활용하여 작업을 한다. 몽고디비는 불편하지만 성능면에서 뛰어나다. 대부분의 클라우드 서비스에서 지원을 해준다. 컬렉션(테이블비슷)->다큐먼트(한개의로우비슷)->그 안에 자바스크립트 객체가 들어갈 수 있다. 첫번째 : mongodb 설치 두번째 : path 경로 설정 세번째 : 사용자 경로에 /database/local 추가 네번째 : cmd 창 2개를 열고 하나는 mongod --dbpath C:\Users\lisha\database\local 명령 후 엔터 (몽고db서버라고 생각하기) 다섯번째 : 다른 cmd 창으로 들어가서 mongo -> use local -> db.users.insert({name:'코로나무섭다',age:25}) 분석 : db에 추가할건데~ users라는 콜렉션에(테이블) 추가할거야~ users는 만든적이 없지만 db자체에서 그냥 만들어버린다고 한다. 여섯번째 : 검색 -> db.users.find().pretty()를 하면 댐 -> 자동으로 _id가 생성
